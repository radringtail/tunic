By far the longest category, 100% (Restricted) collects all major completion elements in the game, farms for bits and spends upgrades for several minutes...and allows no skips or glitches throughout.

100% Restricted is a strange run. It builds on the same skeleton as the other, shorter Restricted runs, making small detours for convenient items up through the Relics. Once daylight returns, the run revisits many areas with Laurels to complete the checklist. But.

There are just over 3000 bits in every chest in the game. If you collect and use all 12 effigies, that's another ~3600 bits. Unfortunately, to use every stat upgrade in the game and purchase the required flasks, you need around 7700 bits. And that doesn't even account for manually sacrificing all 26 upgrade items. At around 90 minutes, it's a long run.

If you're looking for a run without any glitches, with strict rules for routing, and that contains every Restricted strat, look no further!

Be sure you have read over Getting Started before beginning this guide.
