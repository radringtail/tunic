A TUNIC tour de force, 100% (Unrestricted) is a tightly-routed, glitch-heavy, complex run that visits every area in the game as fast as possible using every trick in the book.

Every: upgrade, equipment, relic, page, fairy, treasure, flask, trinket, and slot. Get everything you can on first-visit to most areas, no matter how risky. To avoid having to pay for all those upgrades: steal every upgrade type. Skulls in the swamp? Yep. Only one chance at Scav? Uh-huh. The Fez fairy? Yeah. Forget even one item? Run invalid. But no Golden Path or Heir, I guess?

Aside from the brutal beginning and enormous checklist, 100% Unrestricted is a doable run for only the most committed TUNIC speedrunner.

Be sure you have read over Getting Started before beginning this guide.
