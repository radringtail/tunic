Drop Keys (IRC)

Use ice-rod cancel to drop all three (3) keys together

  • Approach any key slot and drop key
  • Immediately trigger ice-rod, run to the next slot, drop another key
  • Immediately trigger ice-rod, run to the final slot, and either:
    1. Throw a bomb, drop the final key, and death warp, or
    2. Drop the final key, immediately pause, quit, and continue, then throw a bomb for death warp
